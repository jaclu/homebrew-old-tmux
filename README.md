# Jaclu Old-tmux

## How do I install these formulae?

`brew install jaclu/old-tmux/<formula>`

Or `brew tap jaclu/old-tmux` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "jaclu/old-tmux"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
